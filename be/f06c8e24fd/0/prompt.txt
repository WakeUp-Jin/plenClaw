    66|将 `InternalTool` 从 dataclass 升级为更完整的工具定义：
    67|
    68|```python
    69|@dataclass
    70|class ToolParameterSchema:
    71|    type: str = "object"
    72|    properties: dict[str, Any] = field(default_factory=dict)
    73|    required: list[str] = field(default_factory=list)
    74|
    75|@dataclass
    76|class ToolResult:
    77|    success: bool
    78|    data: Any = None
    79|    error: str | None = None
    80|
    81|@dataclass
    82|class InternalTool:
    83|    name: str
    84|    category: str
    85|    description: str
    86|    parameters: ToolParameterSchema
    87|    handler: Callable[[dict[str, Any]], Awaitable[ToolResult]]
    88|    render_result: Callable[[ToolResult], str] | None = None
    89|    is_read_only: bool = False
    90|    should_confirm: bool | None = None
    91|```
