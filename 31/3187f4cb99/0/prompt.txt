    50|# Context 模块重构计划
    51|
    52|## 背景与目标
    53|
    54|当前 `src/core/context` 模块较为简单：一个薄 BaseContext 抽象、一个 ContextManager 混合了消息管理/压缩/持久化逻辑、ChatStore 耦合了 JSONL 实现。重构目标是引入结构化的上下文管理体系，提升可扩展性和可维护性。

    56|## 核心设计决策
    57|
    58|### 1. ContextItem 富数据结构
    59|
    60|系统内部流转使用 `ContextItem` 而非原始 dict，最终通过 `to_message()` 转换为 LLM API 需要的 message 格式。
    61|
    62|```python
    63|@dataclass
    64|class ContextItem:
    65|    role: str                    # "system" | "user" | "assistant" | "tool"
    66|    content: str | None = None
    67|    source: str = ""             # 来源标识: "system_prompt", "memory", "history", "tool"
    68|    priority: int = 2            # 1(LOW) ~ 4(CRITICAL)
    69|    created_at: float = 0.0     
    70|    token_estimate: int = 0     
    71|    metadata: dict = field(default_factory=dict)
    72|    # tool 相关
    73|    tool_calls: list = field(default_factory=list)
    74|    tool_call_id: str | None = None
    75|    name: str | None = None
    76|
    77|    def to_message(self) -> dict[str, Any]:
    78|        """转换为 OpenAI chat message 格式"""
    79|```
    80|
    81|所有上下文模块内部存储 `ContextItem`，BaseContext 的 `format()` 返回 `list[ContextItem]`，ContextManager 最终统一调用 `to_message()` 转换。

   105|### 3. BaseContext 泛型基类
   106|
   107|改造现有 `BaseContext`，支持泛型存储项，提供统一的 CRUD 操作：
   108|
   109|```python
   110|class BaseContext(ABC, Generic[T]):
   111|    def __init__(self):
   112|        self._items: list[T] = []
   113|    
   114|    def add(self, item: T) -> None
   115|    def get(self, index: int) -> T | None
   116|    def get_all(self) -> list[T]
   117|    def clear(self) -> None
   118|    def remove_last(self) -> T | None
   119|    def replace(self, items: list[T]) -> None
   120|    def count(self) -> int
   121|    def is_empty(self) -> bool
   122|    
   123|    @abstractmethod
   124|    def format(self) -> list[ContextItem]:
   125|        """子类实现：将存储项格式化为 ContextItem 列表"""

   128|### 4. SystemPromptContext (分段式设计)
   129|
   130|支持多个 prompt segment，每段有 ID、优先级、内容。核心段由系统定义，动态段可由外部注册。
   131|

   133|@dataclass
   134|class PromptSegment:
   135|    id: str                # 唯一标识: "core", "tools_desc", "user_custom", "memory_inject"
   136|    content: str
   137|    priority: int = 5      # 越大越靠前
   138|    enabled: bool = True
   139|
   140|class SystemPromptContext(BaseContext[PromptSegment]):
   141|    def register_segment(self, segment: PromptSegment) -> None
   142|    def update_segment(self, segment_id: str, content: str) -> None
   143|    def remove_segment(self, segment_id: str) -> None
   144|    def enable_segment(self, segment_id: str) -> None
   145|    def disable_segment(self, segment_id: str) -> None
   146|    def format(self) -> list[ContextItem]:
   147|        """按优先级排序，合并为一条 system ContextItem"""
